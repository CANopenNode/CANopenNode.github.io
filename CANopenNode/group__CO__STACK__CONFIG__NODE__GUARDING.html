#define CO_CONFIG_NODE_GUARDING   (0)

Configuration of Node Guarding CANopen Node Guarding, an older alternative to the Heartbeat protocol..

Possible flags, can be ORed:

  • CO_CONFIG_NODE_GUARDING_SLAVE_ENABLE - Enable Node guarding slave.
  • CO_CONFIG_NODE_GUARDING_MASTER_ENABLE - Enable Node guarding master.
  • CO_CONFIG_FLAG_TIMERNEXT - Enable calculation of timerNext_us variable inside CO_nodeGuardingSlave_process().
